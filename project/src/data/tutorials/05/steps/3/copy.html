Making it sortable (part 1)

It's time to make our table sortable. We've added a 'sortable' class to the three headers to indicate they can be clicked on.

First, let's add a proxy sort event to each column header, specifying the column header as an event argument:

<th class='sortable' on-tap='sort:name'>Superhero name</th>
<th class='sortable' on-tap='sort:realname'>Real name</th>
<th class='sortable' on-tap='sort:power'>Superpower</th>

That way, when the user taps one of the column headers, the view will fire a sort event.

ractive.on( 'sort', function ( event, column ) {
  alert( 'Sorting by ' + column );
});

Execute the code. When you tap on the three sortable headers, the browser should alert the name of the column we're sorting by. Now we just need to add the sorting logic.
